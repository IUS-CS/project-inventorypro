# InventoryPro

An inventory management system that tracks items by category, quantity, transaction history, and location.

## Team

- Dustin De Luna
- Jared Miller
- Nnaemeka Onochie

## Tech Stack

- **Database:** PostgreSQL
- **Backend:** Java + Spring Boot
- **Frontend:** React + TypeScript
- **Barcode Scanning:** ZXing

## Prerequisities:
Make sure you have installed:
- Java
- Maven
## How to test:
- Clone the repository
- Run command git clone https://github.com/IUS-CS/project-inventorypro
- Change directory to the project (cd project-inventorypro)
- Run command (mvn test)
